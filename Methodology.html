METHODOLOGY

Some sub categories of data is missing for certain periods, the data has been approximated by using the parent category data and available child data by averaging missing categories. For example if Meat Poultry and Fish shows 5% inflation, but that only Meat has data showing 3% inflation. Using the relative importance is 33% for each sub category, we can deduce that poultry and fish had 6% inflation

If no special weight has been set in a subcategory, the highest parent will be used for calculating the CPI for limiting approximation errors.

The raw data without Hedonic adjustmenent has been proven difficult to find. Also reversing the adjustment seems like it would lead to wrong result. The only solution for now is too not use the data from the category.

Categories of spender are not defined by BLS, most relevant data only separate by income quintile, Age or home owner vs Renter. The raw data is available but requires heavy work for creating meaningful categories: https://www.bls.gov/cex/pumd_data.htm#csv

The best inflation number will be with custom categories weight, but types of consumers will be based on simplified version of stereotypical consumers.
